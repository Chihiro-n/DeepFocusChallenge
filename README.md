📋 1. コンペティション概要
〜 汎用的な「目の良さ」を手に入れろ！ 〜
「その日の画像のクセに惑わされるな。真のボケを見抜け。」

MLCaT Cupへようこそ！
本コンペティションの課題は、走査型電子顕微鏡（SEM）画像のFocusぼけ（デフォーカス量）を、高精度かつ「汎用的」に推定することです。
SEM画像は、日々の測定条件や対象物によってコントラストや明るさが大きく変動します。ある特定の条件下で学習したモデルが、別の日の画像には全く通用しない――そんな経験はありませんか？

本コンペでは、測定条件の変動（ドメインシフト）に頑健な、真に実用的なフォーカス判定モデルの構築を目指します。

📋 2. 背景
人間の目には明白、AIには難問？
不思議なことに、SEM画像のフォーカスが合っているか、ボケているかは、熟練したエンジニアが見れば一目瞭然です。画像の明るさが変わろうが、コントラストが変わろうが、人間は「ボケている」という事実を直感的に正しく認識できます。

しかし、これを機械学習モデルに行わせると、途端に問題が発生します。
ある条件で完璧に学習したはずのAIが、少し照明条件が変わっただけの画像に対して、全く見当違いな予測値を吐き出すのです。
なぜ、人間の目には不変に見える「ボケ」の特徴が、AIには伝わらないのでしょうか？

既存のモデル（timm等のバックボーン）は、画像のテクスチャやコントラストの強度に過剰に依存している可能性があります。本コンペティションでは、この「謎」を解き明かし、人間の視覚のように条件変動に惑わされず、純粋な光学的ボケだけを見抜くアルゴリズムを求めています。

📋 3. タスク詳細
目的
与えられたSEM画像が、**ベストフォーカス位置からどれだけずれているか（デフォーカス量）**を回帰予測してください。

データの性質
多様な条件: 学習用データとして、様々な日、様々な条件で撮影された画像が提供されます。
未知の条件: テストデータには、学習データに含まれない条件（ドメイン）の画像が含まれる可能性があります。見た目の画質変動に惑わされず、純粋な「像のボケ」だけを抽出する特徴量設計が求められます。
ヒント
学習データとして提供されるのは主に「ベストフォーカス（abs_focus=0)」の画像です。参加者は以下のアプローチなどを検討する必要があります。

ボケの合成 (Simulation): ベストフォーカス画像に対し、ガウシアンブラーやレンズの点広がり関数（PSF）等を適用し、擬似的にボケ画像を作成して学習させる。
リファレンスの活用: sampleフォルダに提供される基準データを手掛かりに、モデルのキャリブレーション（補正）を行ってください。
📋 4. 評価指標
本コンペティションでは、モデルの**汎用性（Robustness）**を重視するため、以下の指標を採用します。

Pattern-averaged RMSE (Macro-averaged RMSE)
テストデータに含まれる K 種類の画像パターン（または条件群）ごとにRMSEを計算し、その平均をとります。特定のパターンだけで高精度でも、苦手なパターンがあるとスコアが悪化します。
