# EXP000: 画像→特徴量→表形式で推論するベースライン

深層モデル使わず、古典的なボケ指標を抽出して軽量モデルに入れるシンプル構成です。

## サマリー

| 指標 | 値 |
|------|-----|
| CV RMSE | 20.46 |
| LB Score | 30.05 |
| 特徴量数 | 3 |
| モデル | Ridge回帰 |

## パイプライン概要

1. **特徴量抽出（画像→表データ）**  
   - Laplacian分散: 高周波が減るほど小さくなる  
   - FFT帯域比（low/mid/high）: ボケると高周波比率が低下  
   - Sobel勾配統計: エッジが弱くなるほど平均・最大が下がる  
2. **スケーリング**: 標準化してスケール差を除去  
3. **回帰**: Ridgeでベースラインを構築  
4. **推論**: 各画像を単独で特徴抽出し、表データとして予測（テスト画像同士で集約・クラスタリングはしない）

## 主要な学び

- **パターン内相関を重視**: 全体相関が高くても、撮像条件パターン別では無効な特徴がある。パターンごとの一貫性が重要。  
- **物理的に意味のある指標が効く**: 勾配・周波数系は輝度変動に強く、ドメイン差に比較的ロバスト。  
- **シンプルでも強いベースライン**: 3特徴量 + 線形回帰で、ボケ方向の符号が揃った指標のみを使うことで安定。
